Cohen's κ — station-level categorical-agreement matrix. Built on ~285 IMD weather stations, each matched to the nearest SHRAM grid point (≤30 km). Both sides reflect today's status: IMD station today's observed heatwave flag (HW=1 or Severe-HW=2), SHRAM the live zone at the paired grid point. κ=1 perfect, κ=0 chance, κ<0 worse than chance. Buckets: <0.2 poor · 0.2–0.4 fair · 0.4–0.6 moderate · 0.6–0.8 substantial · 0.8+ near-perfect. Hover any cell for the underlying 2×2 confusion counts.

Slice by region: use the Station-type filter at the top of the page (Plain · Coastal · Hilly) to restrict the κ calculation to one or more region types. The active filter is reflected in the n-counts on each cell tooltip. Note: the IMD station feed only carries a Heat-Wave flag (no Hot Day / Warm Night codes), so the lower rows are blank.

Cohen's κ — district-level forecast agreement. Built on ~764 IMD districts × matching SHRAM district forecasts. For the selected forecast day, rows are IMD warning codes (Heat Wave / Hot Day / Warm Night) at cumulative severity tiers (Watch+ / Alert+ / Warning) from the WFS feed, columns are SHRAM day-peak zones at progressively higher thresholds. Same κ interpretation as the today matrix; this one has signal because IMD typically flags many districts for the day ahead even on quiet days.
